# Feedback App

## About This App

This is a react app to leave feedback for a product or service

## Features of this app

- Service Review with rating
- Review Count
- Average Rating Count
- Delete Rating
- Edit Review and Change Rating
- Update Rating
- About Page with Route
- Responsive Design

## Used Technologies

- HTML
- CSS
- JS
- React
- React Router DOM
